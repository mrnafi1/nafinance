# 🚀 NaFinance - Advanced Personal Finance Tracker

NaFinance is a modern, feature-rich, and highly secure personal finance management application built with React. It helps you track your income, expenses, wallets, debts, and savings goals seamlessly with a beautiful, responsive UI.

## ✨ Features

* **📊 Dashboard & Analytics:** Real-time tracking of total balance, today's spending/income, and dynamic visual charts (Pie & Bar charts using Recharts).
* **💰 Wallet Management:** Manage multiple wallets (Cash, Bank, etc.), transfer money between wallets, and perform quick top-ups.
* **📝 Transaction Tracking:** Add, edit, and delete transactions. Prevent negative balances with smart validation.
* **🤝 Debts & Loans:** Keep track of money you owe or are owed, with borrow and return dates.
* **🎯 Planning & Goals:** Set multi-savings goals (e.g., PC Build, Bike) and track progress. Set monthly budget limits with smart 80% usage alerts.
* **🔁 Auto-Recurring Bills:** Set up recurring transactions that automatically apply on specific dates.
* **🔒 Security:** 4-digit PIN Lock system to keep your financial data private.
* **🎨 Customization:** * Add Custom Categories with custom emojis and colors.
  * Smooth Dark/Light theme transition.
  * Bilingual Support (English & Bengali).
* **💾 Data Safety:** Entirely offline with LocalStorage persistence. Download backups as JSON and restore them anytime.
* **📤 Export & Reports:** Export data as CSV, print as PDF, or download a high-res PNG screenshot of your dashboard.

## 🛠️ Tech Stack

* **Frontend:** React (Vite)
* **Icons:** Lucide-React
* **Charts:** Recharts
* **Export:** HTML2Canvas
